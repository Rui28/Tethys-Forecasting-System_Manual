1                Introduction

1.1             System requirements

The Tethys Forecasting system does not require installation and can run from the browser. It works best with Firefox, Chrome or Edge web browsers.

A reliable and relatively fast internet connection is essential for a good used experience because large amounts of data are transferred between the server and the client – particularly in the map or dashboard view.

1.2             Changing language preferences

The Tethys Forecasting System presently works in:

·         English

·         Portuguese

The translation is automatically done in the user’s web browser. To change the language of Tethys the options of the web browser should be changed accordingly.

1.3             Changing text ang figure sizes

As most web pages, the text and figure size of the Tethys Forecasting System can be easily adjusted. To do so, it suffices to press CTRL and rolling the mouse wheel up (increase the size) or down (decrease the size).

Figure 1.7:   Main page at different display sizes: 70%, 100% and 150%.

1.4             Logging in / out

 

Access to maps, data, and other restricted functionalities is only possible to registered users. All access must be provided by one of the administrators of the website.

If you already have an account with adequate access privileges, you should login. This can be done on the top right corner of the main page or in a prompt that will appear when you try to access any restricted functionality. Refreshing your browser may be needed after logging in.

After a period without activity Tethys will automatically logoff the user. Logoff can also be done manually on the top right corner of the site.

Figure 1.8:   Login.

 

2                Main parts of the site

2.1             The main page

2.1.1        Contents

This page is entry to the site and from where all functionalities can be accessed (Figure 2.1). It is comprised by a header, a footer, a left panel for navigation, and a main window that is used to display information.

The left panel has three major groups of buttons. The first points towards sections of the initial page. The second loads additional content. The third opens the system management area in another tab.

Figure 2.1:   The main page.

2.1.2        Toggling top, bottom and left panels

In smaller screens the top, bottom and left margins may take considerable space, impairing the user’s experience. These three areas can be hidden by clicking the buttons that appear when the mouse passes above them (Figure 2.2). When the panels are hidden the buttons move to the edge of the window and remain visible. Another click will again show the corresponding panel.

Figure 2.2:   How to hide the top, bottom and left panels for extra room on the main window. Hoovering the mouse over any of these areas will make the buttons appear. Clicking once will hide the corresponding area.

2.2             The map

The map presents a convenient way to visualize and access data. Users can interact with a variety of background maps, georeferenced layers, time series of different types, and raster layers.

All is controlled by the tree menu that appears on the top right. More information on the map can be found in section 4.

Figure 2.3:   The map view.

2.3             The dashboard

The dashboard provides a customizable overview of the system. It contains four main fields, each of them capable of displaying a large amount of information:

A.    An animated map view.

B.    Alarms.

C.    Plots.

D.    Tables.

More information about the dashboard and its four components can be found in section 4.

Figure 2.3:   The dashboard.

2.4             The system management

The system management allows accessing all the configurations of the Tethys Forecasting System. It is here that now users are introduced, the data is updated, and the forecasting models controlled. A thorough view of the functionalities it contains is presented in section 5

Figure 2.4:   The system administration page.
